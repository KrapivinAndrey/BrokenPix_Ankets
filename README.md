# BrokenPix Robotics - Сайт-визитка команды

Сайт-визитка команды по робототехнике в стиле киберпанк. Каждая страница содержит информацию об одном участнике команды.

## Технологии

- **Nuxt 3** - фреймворк для Vue.js
- **Vue 3** - прогрессивный JavaScript фреймворк
- **TypeScript** - типизированный JavaScript
- **Tailwind CSS** - утилитарный CSS фреймворк

## Установка

1. Установите зависимости:
```bash
npm install
```

## Запуск

Запустите сервер разработки:

```bash
npm run dev
```

Сайт будет доступен по адресу: `http://localhost:3000`

## Навигация

- Откройте страницу участника: `http://localhost:3000/1`, `http://localhost:3000/2` и т.д.
- Используйте кнопки "Предыдущий" / "Следующий" для навигации между участниками
- Используйте стрелки влево/вправо на клавиатуре для навигации

## Структура проекта

```
├── app.vue                 # Корневой компонент приложения
├── components/
│   └── TeamMember.vue     # Компонент карточки участника
├── pages/
│   └── [id].vue           # Динамическая страница участника
├── data/
│   └── members.ts         # Данные участников команды
├── types/
│   └── member.ts          # TypeScript типы
└── tailwind.config.js     # Конфигурация Tailwind CSS
```

## Участники команды

В файле `data/members.ts` хранятся данные всех участников. Каждый участник содержит:
- ФИО
- Возраст
- Хобби
- Во что играешь
- Как провел лето
- Ожидания от команды

## Стиль

Сайт выполнен в стиле киберпанк с использованием:
- Темных фонов (#0a0a0a, #1a1a2e)
- Неоновых акцентов (синий #00f0ff, розовый #ff00ff, зеленый #00ff41)
- Анимированных эффектов свечения
- Градиентных границ и теней
